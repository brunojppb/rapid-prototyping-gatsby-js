Static Site Generators come to the rescue

Ok, I don't want to become a DevOps Engineer. I just want a website.
You basically have those frameworks that can generate static pages for you, without going throught the hustle of caring about infrastructure and so on.

What is Gatsby

  1. Static site generator based on React.
  2. You write your React components just as usual and Gatsby generates static pages out of it. You still get all the benefits from React like state management and hooks.
  3. All the benefits from static sites/server-side rendering like SEO and speed.
